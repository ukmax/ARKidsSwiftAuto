Подготовка проекта 

Чтобы собрать проект вам потребуется:

1. Установить gem CocoaPods
2. Установить библиотеки через CocoaPods


Приступим ...

Установка CocoaPods 

1. Открыть терминал 
2. Перейти в папку с проектом cd .. /TinkoffFintech
3. Выполнить $ sudo gem install cocoapods
4. Прописать глобальные переменные 

export GEM_HOME=$HOME/.gem
export PATH=$GEM_HOME/bin:$PATH



Установить библиотек 
1. Открыть терминал 
2. Перейти в папку с проектом cd .. /TinkoffFintech
3. Выполнить pod install

Готов, можно собирать проект
